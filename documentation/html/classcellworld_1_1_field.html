◆ updatePositions()

void cellworld::Field::updatePositions ( )

Обновляет позиции существ.

Ввод данных в нейросеть. Происходит вся работа нейросетей. Забирает энергии у существ за передвижение. Оставляет на предыдущей позиции остатки энергии. Обрабатывет коллизию существ. Работает в многопоточном режиме, без инициализаций.

◆ updateStates()

void cellworld::Field::updateStates ( )

Обновляет состояние существ. Если у живого существа <0 энергии - оно умирает. Если существо хочет размножится, то одно из сосодних существ становится живым, с геном, наследуемым от предка. Работает в многопоточном режиме, без инициализаций.
