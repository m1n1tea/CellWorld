\doxysubsubsection{Подробное описание}
Является синглтоном. Содержит набор функций, которые представляют собой отдельные сцены. Сцены самостоятельны внутри себя(за исключением записи и чтения данных, которые доступны для всех сцен), должны предоставлять возможность сменить сцену.
